# Support Vector Machine

### Objective
Build SVM models with different kernels to predict drug response.

### Dataset
Drug Response dataset — 500 records, 5 clinical features, binary target.

### Contents
- 4 kernel comparison (linear, poly, rbf, sigmoid)
- ROC curve and confusion matrix
- Standard scaling (required for SVM)
- Drug Response prediction for personalized medicine

### How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
