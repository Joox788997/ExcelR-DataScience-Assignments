# Hypothesis Testing — Bombay Hospitality

### Objective
Test if actual operating costs exceed the theoretical cost model.

### Dataset
Bombay Hospitality case — sample of 25 weeks, theoretical model W=$1,000+$5X.

### Contents
- One-tailed t-test with formulas
- Step-by-step calculations shown
- Confidence interval
- Business interpretation

### How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
