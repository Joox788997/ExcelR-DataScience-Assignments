# Random Forest Classification

### Objective
Apply Random Forest, Bagging, and Boosting to classify glass types.

### Dataset
Glass Identification dataset — 214 samples, 9 chemical features, 7 glass types.

### Contents
- Random Forest with OOB score
- Bagging and Boosting (AdaBoost, Gradient Boosting) comparison
- Feature importance analysis
- Bagging vs Boosting interview questions

### How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
