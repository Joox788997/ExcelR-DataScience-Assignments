# Clustering Analysis — K-Means & DBSCAN

### Objective
Segment airline loyalty customers using K-Means and DBSCAN.

### Dataset
East-West Airlines — 3,999 customers, 12 features.

### Contents
- K-Means with elbow method and silhouette analysis
- DBSCAN with multiple epsilon values
- Hierarchical clustering dendrogram
- Cluster profiling and interpretation

### How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
