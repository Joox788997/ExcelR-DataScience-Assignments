# Anime Recommendation System

### Objective
Build content-based recommendation using cosine similarity.

### Dataset
Anime dataset — 12,294 anime with genre, type, rating, members.

### Contents
- TF-IDF vectorization on genres
- Cosine similarity matrix
- Recommend function with threshold tuning
- Collaborative filtering interview answers

### How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
