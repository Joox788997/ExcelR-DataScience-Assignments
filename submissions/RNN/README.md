# RNN / LSTM / GRU — Milk Production Forecasting

### Objective
Compare RNN, LSTM, and GRU for monthly milk production forecasting.

### Dataset
168 monthly milk production records.

### Contents
- RNN vs LSTM vs GRU architecture comparison
- Training loss curves
- 12-month lookback sequence
- Forecast visualization for all three models

### How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
