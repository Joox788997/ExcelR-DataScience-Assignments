# Principal Component Analysis

### Objective
Apply PCA for dimensionality reduction on the Wine dataset.

### Dataset
Wine dataset — 178 samples, 13 chemical features, 3 classes.

### Contents
- Scree plot and cumulative explained variance
- 2D PCA visualization with class separation
- Feature loadings on PC1 and PC2
- 90% variance threshold analysis

### How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
