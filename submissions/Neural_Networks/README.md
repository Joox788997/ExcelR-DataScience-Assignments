# Neural Networks — Sonar Classification

### Objective
Build an ANN to classify sonar signals as mines or rocks.

### Dataset
Sonar dataset — 208 signals, 60 frequency features, binary target (M/R).

### Contents
- MLP architecture (60→24→12→6→2) with ReLU
- Training loss curve
- ROC-AUC and confusion matrix
- Cross-validation

### How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
