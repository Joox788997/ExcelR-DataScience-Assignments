# ExcelR Data Science Assignments – Submission Portfolio

## Overview

This repository contains my completed Data Science assignments submitted as part of the ExcelR Data Science Program. The assignments cover a broad range of topics including data preprocessing, statistical analysis, machine learning, deep learning, recommendation systems, time series forecasting, and model deployment.

Each assignment has been organized in a separate folder and includes:

* Fully completed Jupyter Notebook (.ipynb)
* README documentation
* Requirements file
* Supporting resources where applicable

---

## Assignments Included

### 1. Data Transformation

Worked on data preprocessing techniques including handling missing values, encoding categorical variables, feature engineering, normalization, standardization, and data transformations.

### 2. Multiple Linear Regression

Built and compared multiple regression models, performed coefficient interpretation, residual analysis, Ridge Regression, and Lasso Regression.

### 3. Logistic Regression

Developed a classification model on the diabetes dataset including:

* Data preprocessing
* Outlier treatment
* Multicollinearity analysis
* ROC Curve analysis
* Cross-validation
* Streamlit deployment

### 4. Decision Tree

Implemented Decision Tree models with feature importance analysis, hyperparameter tuning, and performance evaluation.

### 5. Random Forest

Built ensemble learning models and compared Random Forest performance with other tree-based techniques.

### 6. Support Vector Machine (SVM)

Explored multiple kernel functions and evaluated classification performance using standard metrics.

### 7. Principal Component Analysis (PCA)

Performed dimensionality reduction and analyzed explained variance and component contributions.

### 8. Neural Networks

Designed and evaluated neural network models for classification tasks using multilayer architectures.

### 9. XGBoost & LightGBM

Compared gradient boosting frameworks and analyzed model performance and feature importance.

### 10. Clustering

Applied K-Means, Hierarchical Clustering, and DBSCAN techniques to identify meaningful customer segments.

### 11. Recommendation System

Built a content-based recommendation engine using similarity measures and recommendation logic.

### 12. Time Series Analysis

Performed trend analysis, stationarity testing, forecasting, and model evaluation on time series data.

### 13. Hypothesis Testing

Conducted statistical hypothesis testing, confidence interval estimation, and business interpretation of results.

### 14. RNN / LSTM / GRU

Implemented deep learning models for sequential forecasting and compared recurrent neural network architectures.
