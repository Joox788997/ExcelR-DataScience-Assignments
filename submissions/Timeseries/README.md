# Time Series — Exchange Rate Forecasting

### Objective
Forecast USD/AUD exchange rates using ARIMA and Exponential Smoothing.

### Dataset
7,588 daily USD to AUD exchange rate observations.

### Contents
- ARIMA(1,1,1) model
- Exponential Smoothing
- ADF stationarity test
- ACF/PACF plots
- Forecast visualization

### How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
