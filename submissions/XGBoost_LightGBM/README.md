# XGBoost & LightGBM Comparison

### Objective
Compare XGBoost and LightGBM for diabetes prediction.

### Dataset
Pima Indians Diabetes dataset — 768 records, 8 features, binary target.

### Contents
- XGBoost vs LightGBM side-by-side
- Feature importance comparison
- Confusion matrices
- Gradient boosting framework evaluation

### How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
